# 🎬 Scroll Animation Studio

A modern one-page landing page focused on smooth scroll-based animations and clean UI design.

This project demonstrates how to use the Intersection Observer API to create professional reveal effects, combined with multiple thresholds and stagger animations for a premium user experience.

---

## 🚀 Live Demo

👉 https://johnyisbackk.github.io/js-scroll-animation-studio/

---

## ✨ Features

* 🔥 Scroll progress bar
* 🎯 Multiple IntersectionObserver thresholds
* 🎬 Reveal animations (up, left, right, scale)
* ⏱️ Stagger animation effect for cards
* 🧊 Sticky header with blur and scroll effect
* 📱 Fully responsive layout
* 🎨 Modern dark UI design

---

## 🧠 What I Learned
