About Studio

Designed to practice modern scroll-based UI animation.

This project is focused on improving animation logic using JavaScript and Intersection Observer. It helps practice reveal effects from different directions while keeping the layout clean and professional.

The goal is not just to create a nice landing page, but to build a reusable animation system for future websites and portfolio work.
